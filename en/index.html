BackgroundThe secret behind LLM is that it will generate tokens one by one based on all the previous tokens. Let’s assume that we have already generated $t$ tokens, denoted by $x_{1:t}$. In the next iteration, the LLM will generate $x_{1:t+1}$. Note that the first $t$ tokens are the same. $$x_{1:t+1}=\text{LLM}(x_{1:t})$$ The next iteration is similar. $$x_{1:t+2}=\text{LLM}(x_{1:t+1})$$ In summary, in each iteration, we will use the output of the previous round as a new input for the LLM.
